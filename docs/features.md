# Travel Lounge Features - Web App

## Public Services

- **Cruises**: Full listing of available sea voyages with dynamic filtering.
- **Flights**: International and regional flight booking options.
- **Hotels**: Premium resorts in Mauritius and Rodrigues with dynamic room pricing.
- **Day Packages**: "Sea" and "Land" activity filtering for local residents and tourists.
- **Group Tours**: Region-based tours (Asia, Europe, Middle East) with itineraries.
- **FAQ Page**: Fully dynamic category-based help section with search.
- **News & Editorial**: Portal for the latest travel trends, destinations, and expert tips.

## User Experience

- **Modern Hero Slider**: Dynamic, admin-managed media with video support and call-to-actions.
- **Interactive Reviews**: Real-time display of moderated guest experiences.
- **Functional Contact Support**: Direct Supabase-integrated inquiry system.
- **Newsletter Subscription**: Real-time signups from both Footer and Homepage.
- **Navigation**:
  - Glassmorphism sticky header with "About" dropdowns.
  - Simplified "Hotels" direct access and professional "Team" profiles.
- **Performance**:
  - Mobile-first approach for all listing and detail pages.
  - Memoized clients and optimized Next.js rendering.
  - Robust error boundaries and loading states for a premium feel.
