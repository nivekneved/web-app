# 01 Overview & Ecosystem

## Project Mission
The **Travel Lounge Web Application** is the flagship guest-facing platform. It is designed to redefine luxury travel through a unified, high-performance booking engine that adheres to "11/10 Elite Standards" in design, speed, and conversion.

---

## Core Functional Capabilities
- **Universal Elite Search**: Multi-module discovery for Hotels, Flights, Activities, and Cruises.
- **Interactive Booking Wizard**: A seamless 4-step reservation experience with real-time tax and pricing validation.
- **Branding Excellence**: Dynamic logo and theme propagation fueled by the Admin site settings.
- **Post-Booking Concierge**: Integration with WhatsApp and Email for immediate post-reservation engagement.
- **Flight IBE Integration**: Advanced native iframe handling for the GOL IBE D4 engine on a dedicated `/flights` page.

---

## Technical Stack
- **Framework**: Next.js 15 (App Router) + React 19.
- **Styling**: Tailwind CSS v4 + Framer Motion for premium animations.
- **Design Tokens**: Standardized on **Outfit Black (900)** and a high-definition **Slate-300** palette.
- **Performance**: Optimized via server-side rendering (SSR) and dynamic image resolution.
- **Backend**: Supabase (PostgreSQL, Real-time, Auth).
